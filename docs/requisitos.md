## Requisitos Funcionais

RF01 – O sistema deve permitir o empréstimo de livros.
RF02 – O sistema não deve permitir empréstimo a usuários com multas pendentes.
RF03 – O sistema deve atualizar o status do livro após empréstimo.

## Requisitos Não Funcionais

RNF01 – O sistema deve seguir princípios SOLID.
RNF02 – O sistema deve possuir testes unitários.
